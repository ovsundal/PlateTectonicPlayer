## Codebase Patterns

### Git Worktree Layout
- **Where**: root of repo
- **Pattern**: The project git repo is at `C:/Users/ove.sundal/WebstormProjects/PlateTectonicPlayer/`. The archon worktree (`archon/thread-ae2b8fff`) is a git worktree of that same repo. All source files must be created in the WORKTREE path (`.archon/workspaces/.../worktrees/archon/thread-ae2b8fff/`), NOT in WebstormProjects directly, so they are on the correct branch for commits.
- **Example**: `git add package.json` from the worktree path stages to `archon/thread-ae2b8fff`

### npm Commands
- **Where**: project root
- **Pattern**: Run `npx tsc --noEmit`, `npx eslint src`, `npx vite build` from the WORKTREE path (cd first). Shell CWD resets between Bash commands so always use `cd /path && command` in a single command.
