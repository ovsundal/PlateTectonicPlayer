## Codebase Patterns

### deck.gl GlobeView Setup
- **Where**: `src/components/Globe.tsx`
- **Pattern**: Import DeckGL from `@deck.gl/react`, GlobeView from `@deck.gl/core`, layers from `@deck.gl/layers`. Pass `views={new GlobeView()}` to DeckGL. Layer instances defined outside component to avoid recreation.
- **Example**: `<DeckGL views={new GlobeView()} initialViewState={...} controller={true} layers={layers} />`

### Global CSS Reset (Vite project without index.css)
- **Where**: `src/index.css`, imported in `src/App.tsx`
- **Pattern**: When no index.css exists, create one at `src/index.css` and import it from App.tsx. Covers `*, html, body { margin:0; padding:0; box-sizing:border-box }` and `#root { width:100vw; height:100vh; overflow:hidden }`.
